# 학점 계산기

평가 항목(중간고사, 기말고사, 과제 등)의 비중과 내 점수를 입력하면 예상 학점을 계산해주는 도구입니다. 절대평가와 상대평가를 모두 지원합니다.

## 기능

- 과목을 여러 개 추가하고 각각 평가 항목(항목명/비중/점수)을 자유롭게 구성
- 비중 합계가 100%가 아니면 경고 표시
- **절대평가**: 가중 평균 점수를 기준으로 학점 산정 (A+ ~ F)
- **상대평가**: 수강 인원과 예상 등수를 입력하면 백분위 기준으로 학점 산정
- 과목이 2개 이상이면 전체 과목 요약 카드 표시

## 구조

단일 HTML 파일(`index.html`)로 구성된 순수 클라이언트 사이드 앱입니다. 별도의 빌드 과정이나 백엔드 없이 브라우저에서 바로 동작합니다.

## 실행

브라우저에서 `index.html`을 열면 바로 사용할 수 있습니다.

```bash
open index.html
```
